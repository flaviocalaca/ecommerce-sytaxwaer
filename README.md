
# SytaxWaer - Tênis Sneakers Online

## Descrição

Este é um projeto de e-commerce de tênis chamado SytaxWaer. O site apresenta uma interface moderna e responsiva, com foco na apresentação de produtos e categorias de forma clara e atraente.

## Funcionalidades

- **Navegação Intuitiva:** Menu principal com links para categorias (Masculino, Feminino, Outlet), lojas e informações sobre a empresa.
- **Seção de Herói:** Banner principal com chamada para ação, destacando um produto ou campanha.
- **Categorias de Produtos:** Seção visual com as principais categorias de tênis (Casual, Esportivo, Moderno, Futurista).
- **Grid de Produtos em Destaque:** Apresentação de produtos selecionados em um layout de grid dinâmico.
- **Rodapé Completo:** Links de navegação, formulário de newsletter e ícones de redes sociais.

## Tecnologias Utilizadas

- **HTML5:** Para a estrutura semântica do site.
- **CSS3:** Para a estilização e layout, utilizando variáveis, flexbox e grid.
- **Design Responsivo:** O site é projetado para ser totalmente responsivo e adaptável a diferentes tamanhos de tela (desktop e mobile).

### Detalhes dos Arquivos

- **`index.html`**: O arquivo principal que contém a estrutura da página.
- **`css/`**: Contém todos os arquivos de estilização.
  - **`reset.css`**: Reseta os estilos padrão do navegador.
  - **`variables.css`**: Define as variáveis de cor e outras propriedades reutilizáveis.
  - **`base.css`**: Estilos básicos para o corpo da página e elementos HTML.
  - **`layout.css`**: Estilos para o layout geral do site.
  - **`components/`**: Contém os arquivos CSS para cada componente da página (cabeçalho, rodapé, etc.).
- **`images/`**: Armazena todas as imagens do projeto, organizadas em subpastas.

## Como Utilizar

Para visualizar o site, basta abrir o arquivo `index.html` em qualquer navegador de sua preferência.

```bash
# Clone o repositório (opcional)
git clone https://github.com/seu-usuario/ecommerce-sytaxwaer.git

# Navegue até o diretório do projeto
cd ecommerce-sytaxwaer

# Abra o index.html no navegador
# (pode ser feito clicando duas vezes no arquivo)
```
